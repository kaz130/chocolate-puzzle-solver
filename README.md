# Polyomino Solver
ポリオミノの箱詰めパズルを Fixstars Amplify で解くプログラム．

事前に Fixstars Amplify のアカウント登録とアクセストークンの取得が必要．

## Requirement
- Python 3.9

## Usage
```sh
echo "TOKEN=access_token" > .env
pip install -r requirements.txt
python -m psolver sample.toml
```
